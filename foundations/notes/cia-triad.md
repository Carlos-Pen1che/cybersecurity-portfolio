# CIA Triad

## Overview

CIA Triad is a model that represents a three core principles of cybersecurity: Confidentiality, Integrity, Availability

## Example

In a collision attack, integrity is compromised when an attacker generates two different inputs with the same hash, which can bypass hash-based integrity checks. This was possible with the MD5 algorithm.

## Why It Matters
This matters because compromised integrity can lead to SPII and PII exposure. Organizations must use modern algorithms such as SHA-256.
